# Python Basics Projects

This repository contains beginner-friendly Python programs built to practice core programming concepts.

## Projects
- **hello.py** – Basic input/output program
- **calculator.py** – Interactive calculator with error handling
- **number_guess.py** – Number guessing game with attempt tracking
- **todo.py** – Terminal-based to-do application with file persistence

## Concepts Covered
- Variables and data types
- Control flow (if/else, loops)
- Functions
- Exception handling
- File I/O
- Git & GitHub workflow
